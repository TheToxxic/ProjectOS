# Project OS — Windows Releases

Dieses Repository enthält ausschließlich die **Windows-Releases** von Project OS
(Installer, Portable-ZIP und das Auto-Updater-Manifest `latest.yml`).

## Download

→ **[Releases](https://github.com/TheToxxic/ProjectOS/releases)** — neueste Version laden:

- `Project.OS.Setup.x.x.x.exe` — Installer (empfohlen)
- `Project.OS-x.x.x-win.zip` — Portable, kein Admin nötig

Installierte Versionen aktualisieren sich automatisch über den eingebauten Updater.
